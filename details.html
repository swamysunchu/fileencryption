Project Description:

The File Encrypter project, developed by Team 3 as part of a Cyber Security Internship, focuses on enhancing cybersecurity for organizations. The project, titled "Secured File Encryption and Decryption," aims to implement secure encryption for files containing sensitive data using SMTP passwords. The encryption process involves generating a unique key using the Fernet cryptography library.

The key, along with the encrypted file and its filepath, is sent via email using SMTP protocols. The project includes features for both encryption and decryption processes. The user-friendly GUI, built with Tkinter, allows users to select files for encryption and decryption, enter email and SMTP details, and seamlessly perform secure file operations. The Details page provides comprehensive information about the project, including project details, developer information, and company details.

The interface features a visually appealing design with a prominent project title, logo, and intuitive buttons for file encryption, decryption, and accessing project details. The project's HTML page serves as a professional and informative platform, showcasing project highlights and developer contributions.

Encryption Process:

  1. Key Generation:
    • A new encryption key (`key`) is generated using `Fernet.generate_key()`. This key is used for both encryption and decryption processes.
  2. File Encryption:
    • The content of the selected file is read in binary mode (`'rb'`).
    • The Fernet encryption algorithm is applied to the file's content using the generated key (`fernet.encrypt(data)`).
    • The encrypted content replaces the original content in the file.
  3. Key Storage:
    • The encrypted key (`encrypted_password`) is generated by encrypting the original key using the Fernet algorithm.
    • This encrypted key is then stored in a file named "key.txt" for later use in the decryption process.

The Fernet encryption scheme is a symmetric key cryptography algorithm, meaning the same key is used for both encryption and decryption. It provides a straightforward and secure way to encrypt and decrypt data. The `cryptography` library provides a high-level interface for Fernet and other cryptographic operations, ensuring secure and efficient encryption processes.
